Projet C

Le but de ce projet est de réaliser un interpréteur de commande qui permet de définir le nombres de tournois, d’enregistrer pour chaque tournoi les résultats de chacun des 127 matchs, d’extraire des résultats les 128 joueuses qualifiées du tournoi, de calculer le nombre de points gagnés par les joueuses dans un tournoi donné, de donner le résultats des matchs d’un tournoi donné pour une joueuse donnée, de tenir à jour une table des joueuses classées avec le cumul des points gagnés dans les 4 derniers tournois au plus et de donner le classement.

Projet c++

Le but de ce projet est de réaliser un programme en C++ qui permet de jouer au démineur avec 5 opérations élémentaires qui sont : 1. Produire un problème à partir du nombre de lignes, de colonnes et de mines. 2. Produire une grille à partir d’un problème et d’un historique de coups. 3. Déterminer si une partie est gagnée à partir d’un problème et d’un historique de coups. 4. Déterminer si la partie est perdue à partir d’un problème et d’un historique de coups. 5. Produire un nouveau coup à partir d’une grille.
